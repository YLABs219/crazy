《瑞安 我又来了》— 微信打开指南
================================

【重要】微信里不能打开电脑上的 html 文件本身，
必须先把游戏放到「HTTPS 网址」上，再把链接发到微信。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法一：Gitee _pages（国内推荐，免费）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 https://gitee.com 注册/登录
2. 右上角「+」→ 新建仓库
   - 仓库名：ryan-game（随意）
   - 选「公开」
3. 进入仓库 → 「上传文件」
   - 把 zip 解压后的全部内容上传，包括：
     index.html
     游戏 文件夹（含所有 jpg 和 manifest.json）
4. 仓库 → 服务 → Gitee Pages → 启动
5. 等待 1～3 分钟，得到网址，形如：
   https://你的用户名.gitee.io/ryan-game/
   或带 index.html：
   https://你的用户名.gitee.io/ryan-game/index.html
6. 复制链接 → 微信发给孙高敏 → 点开即玩

【私密链接格式】部署完成后，请发带访问码的链接：
  https://你的网址/index.html?key=0506
首次打开会自动记住访问码（同一次浏览器会话内有效）。
也可在页面输入访问码：0506

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法二：Netlify 拖拽上传（简单）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 https://app.netlify.com/drop
2. 把整个 surprise-box 文件夹拖进去
3. 获得 https://xxx.netlify.app 链接
4. 发私密链接到微信：
   https://xxx.netlify.app/index.html?key=0506

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法三：腾讯云 / 阿里云 静态网站（稳定）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

把 zip 解压后整个文件夹上传到对象存储/静态网站，
开启 HTTPS，用分配的域名访问 index.html。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
发微信时注意
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 必须是以 https:// 开头的完整链接
✓ 连同「游戏」文件夹一起上传（任务二图片在里面）
✓ 每人手机会自动存档，清除微信缓存可能丢档
✗ 不要只发 index.html 单个文件（图片会丢失）
✗ 不要用 file:// 或本地路径

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
上传文件清单（缺一不可）

电脑本地测试（同一 WiFi 下手机也能测）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 双击 index.html 应能正常开机（输入访问码 0506）
2. 若双击不行：用 Chrome/Edge 打开，Ctrl+F5 强制刷新
3. 同一 WiFi 下可用局域网地址测试（需本机开简易服务器）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

上传完成后，用手机浏览器（或微信）打开：
  https://你的网址/index.html?key=0506
走一遍：开机 → 开始闯关 → 任务一 → 任务二猜图 → 任务三寻宝

如有问题，检查「游戏」文件夹是否上传成功。
