看图猜店名 - 图片说明
========================

1. 把店铺照片放进本文件夹（与 index.html 同级的「游戏」文件夹）
2. 文件名（不含 .jpg/.png 后缀）就是正确答案
   例如：阿林粉店.jpg → 答案填「阿林粉店」
3. 修改 manifest.json，按顺序列出图片文件名（扩展名必须与实际文件一致）
4. 同时在 index.html 里搜索 quiz → images，保持与 manifest 一致
5. 答对 5 题即可通关（共 7 张图，按顺序作答）

注意：
- 扩展名要用 .jpg（不是 .svg）
- manifest.json 最后一行不要加逗号
- 请双击打开 surprise-box 文件夹里的 index.html（不要移动图片到其他位置）
